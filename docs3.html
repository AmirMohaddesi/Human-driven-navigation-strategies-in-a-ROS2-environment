Customization Guide

Adjusting Robot Count and Initial Positions

You can easily modify the number of robots and their initial positions by editing the launch file located at multi_agent_demo.launch.py. Look for a configuration section similar to this:

robots = [
    {'name': 'tb3_0', 'x_pose': 0.0, 'y_pose': 0.0},
    {'name': 'tb3_1', 'x_pose': 2.0, 'y_pose': 0.0},
    # Add more robots as needed
]

SLAM Toolbox Configuration

To customize SLAM parameters, edit the YAML file found in the params/ directory, for example slam_params.yaml:

map_resolution: 0.05
use_scan_matching: true
loop_closure_enabled: true

Navigation (Nav2) Configuration

For customizing Nav2 behavior, modify parameters in nav2_params.yaml located in the params/ directory:

planner:
  tolerance: 0.1
controller:
  max_velocity: 0.25
  min_velocity: 0.05

Environment and World Setup

Change the Gazebo simulation world by updating the launch argument or directly editing the launch file to specify the world file:

ros2 launch disaster_response_swarm multi_agent_demo.launch.py world:=

Best Practices for Configuration Management

  • Keep your custom configurations version-controlled using Git.
  • Document all changes clearly within YAML and launch files.
  • Make incremental adjustments and test regularly to easily identify issues.
