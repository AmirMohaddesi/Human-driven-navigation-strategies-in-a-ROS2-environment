Getting Started

Cloning and Building the Repository

Follow these steps to clone and build the platform:

mkdir -p ~/ros2_ws/src
cd ~/ros2_ws/src
git clone https://github.com/AmirMohaddesi/disaster_response_swarm.git  # Replace with your repository URL
cd ~/ros2_ws
colcon build --symlink-install
source install/setup.bash

Launching a Multi-Agent Example

To run a basic multi-agent simulation:

ros2 launch disaster_response_swarm multi_agent_demo.launch.py

Visualization with RViZ

Launch RViZ to visualize the robots and their sensor data:

rviz2

In RViZ, manually add the following displays for comprehensive visualization:

  • Robot Model
  • TF
  • LaserScan
  • Map

Ensure all displays are correctly configured for each robot namespace (e.g., /tb3_0).
